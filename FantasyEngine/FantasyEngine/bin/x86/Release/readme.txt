Controls
----------
Arrows       : Move cursor and the character.
Enter        : Accept.
Backspace    : Cancel/Return.
Page Up/Down : Scroll the CTB Window.

Special debug commands
B            : Debug button to call a battle.
Esc          : Force quit.
F12          : Show the FPS.
F9           : Mute the music.
P            : Show position on the map.
