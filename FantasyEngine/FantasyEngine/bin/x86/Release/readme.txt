Controls
----------
Arrows       : Move cursor and the character.
Enter        : Accept/Interact.
Backspace    : Cancel/Return.
Page Up/Down : Scroll the CTB Window.
Esc          : Main menu.
C            : Character Screen.
E            : Equipment Screen.
I            : Inventory Screen.
F9           : Mute/Unmute the music.
K            : Skill Screen.

Special debug commands
B            : Debug button to call a battle.
<            : Force quit.
F12          : Show the FPS.
P            : Show debug informations.
L            : Level up.
V            : Toggle Battle/No Battle
